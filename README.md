# 💧 Hydromart - Water Purifier E-Commerce Website

Hydromart is a responsive e-commerce web application built for purchasing water purifiers online. The project is developed using modern frontend technologies with a mock API for data handling.

## 📌 Features

- 🛒 Browse and view water purifiers
- 🔍 Product listing with details
- 🧾 Add to cart functionality
- 👤 User interface for customers
- 🔐 Admin panel layout for product management
- ⚡ Fast and interactive UI using React

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **Bootstrap**
- **JavaScript (ES6+)**
- **ReactJS**
- **MockAPI (for backend simulation)**

---

## 🔗 API Used

- MockAPI: https://mockapi.io/
- Used for handling product data and simulating backend operatio

This project was built to strengthen frontend development skills using ReactJS and to understand real-world e-commerce application structure including UI design, routing, layouts, and API integration.
